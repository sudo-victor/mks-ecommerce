## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sudo-victor/mks-ecommerce
   ```
2. Install NPM packages
   ```sh
   npm install
   or
   yarn
   ```

### Run

```sh
npm run dev
or
yarn dev
```
